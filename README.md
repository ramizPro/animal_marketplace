Animal Marketplace

Animal Marketplace je spletna platforma za nakup in prodajo živali. Namen aplikacije je povezati prodajalce, rejce in kupce na enem mestu ter zagotoviti varno, pregledno in uporabniku prijazno izkušnjo.

Funkcionalnosti

Pregled živali po kategorijah 

Filtriranje 

Dodajanje oglasov za prodajo ali posvojitev

Registracija in prijava uporabnikov

Nalaganje slik živali

Varnost in zaščita osebnih podatkov
